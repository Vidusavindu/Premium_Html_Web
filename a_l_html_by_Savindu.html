Architecture

DOM ගස (Document Object Model)

බ්‍රවුසරය HTML කියවන්නේ ගහක අතු විහිදෙනවා වගේ ව්‍යූහයකට. මෙය Hierarchy එකක් ලෙස හැඳින්වේ. සෑම Tag එකක්ම Node එකකි.

  • <html>
    Root Element
    • <head>
      Metadata
      • <title>
        Page Name
      • <meta>
        SEO/Viewport
      • <style>
        Internal CSS
    • <body>
      Visible Content
      • Semantic Tags
        • <header>
        • <nav>
        • <footer>
      • Block Elements
        • <div>
        • <p>
        • <table>
      • Inline Elements
        • <a> (Anchor)
        • <span>
        • <img>
The Syllabus

A/L අච්චු පොත 📚

HTML හි සියලුම Elements සහ Attributes විභාග මට්ටමට ගැළපෙන ලෙස මෙහි අන්තර්ගත කර ඇත.

Root & Metadata

වෙබ් පිටුවක පදනම. මේවා Document Object එක හැඩගස්වයි.

  • <!DOCTYPE html> HTML version එක බව Browser එකට කියන Declaration එක.
  • <html lang="en"> ප්‍රධාන Root Element එක. lang කියන්නේ Attribute එකක්.
  • <meta charset="UTF-8"> සිංහල අකුරු වගේ දේවල් Browser එකට තේරුම් ගන්න දෙන Character Encoding එක.

Text Formatting

අකුරු හැඩසැකසීම සඳහා භාවිතා කරන Tags.

  • <h1> - <h6> : Headings (මාතෘකා)
  • <p> : Paragraph (ඡේදයක්)
  • <br> : Line Break (නව පේළියකට යෑම. මෙය Empty Element එකකි)
  • <hr> : තිරස් ඉරක් (Horizontal Rule)
  • <strong><b> : වැදගත් බව පෙන්වීමට අකුරු තද (Bold) කිරීම. (<b> ට වඩා Semantic අගයක් ඇත)
  • <em> <i> : Italic (ඇල කිරීම).
  • <sup> / <sub> : X2 (Superscript) සහ H2O (Subscript).

Lists & Links

Hyperlinks <a href="url" target="_blank">Click</a> href මගින් link එක යන තැනද, target="_blank" මගින් අලුත් Tab එකක open වීමද සිදු කරයි. මෙය Inline Element එකකි.
Lists
  • ul - Unordered List (තිත් සහිත)
  • ol - Ordered List (අංක සහිත)
    (Attribute: type="A", start="5")
  • li - List Item (ලැයිස්තුවේ අංග)
  • dl, dt, dd - Description Lists (විස්තරාත්මක ලැයිස්තු).

Multimedia & Semantic HTML5

Media Elements (රූප සහ ශබ්ද)

  • <img src="pic.jpg" alt="Description"> src = Source Path, alt = පින්තූරය load නොවුණොත් පෙන්වන Alternative Text එක.
  • <audio controls> <source src="song.mp3"> </audio> controls attribute එක මගින් Play/Pause බොත්තම් පෙන්වයි. (Video සඳහා <video> භාවිතා වේ).

Semantic Tags (අර්ථවත් ටැග්)

නිකම්ම <div> දානවා වෙනුවට Search Engines (Google) වලට තේරෙන විදිහට කේතය ලිවීම SEO සඳහා වැදගත් වේ.

<header> <nav> (Navigation) <main> <section> <article> <footer>

Tables (වගු)

විභාගයට අනිවාර්යයෙන් එන Block-level Element එකකි.

  • table - මුළු වගුවම
  • tr - Table Row (තිරස් පේළියක්)
  • th - Table Header (මාතෘකා කොටු - තද අකුරු)
  • td - Table Data (දත්ත කොටු)

A/L Special Attributes:

colspan="2" : තිරස් අතට (Column) කොටු 2ක් එකතු කිරීම.

rowspan="3" : සිරස් අතට (Row) කොටු 3ක් එකතු කිරීම.

Forms (පෝරම)

පරිශීලකයාගෙන් දත්ත (User Input) ලබා ගැනීමට භාවිතා කරන Interactive Elements.

<form action="submit.php" method="POST">
  • <input type="text"> සාමාන්‍ය පෙළ
  • <input type="password"> මුරපද (****)
  • <input type="radio"> එකක් පමණක් තේරීමට (Must share same name attribute).
  • <input type="checkbox"> කිහිපයක් තේරීමට
  • <select> & <option> Drop-down ලැයිස්තු

CSS (Cascading Style Sheets)

HTML මගින් සැදූ web-pageය ලස්සන කරන්නේ CSS වලිනි. මෙහිදී Selectors මගින් HTML tag එක තෝරාගෙන Property: Value; ලෙස styles ලබා දේ.

CSS යොදන ක්‍රම 3 (Methods of applying CSS)

  1. Inline CSS: අදාළ tag එක ඇතුළතම style="..." ලෙස ලිවීම.
    Highest priority. (අවම වශයෙන් භාවිතා කළ යුතුයි - Bad Practice).
  2. Internal CSS: <head> කොටස තුළ <style> tag එකක් දමා ලිවීම.
    එක පිටුවක් පමණක් Style කිරීමට.
  3. External CSS: වෙනම .css ගොනුවක් සාදා <link rel="stylesheet" href="style.css"> ලෙස සම්බන්ධ කිරීම.
    හොඳම ක්‍රමය (Best Practice).
/* CSS Selectors (තෝරාගන්නා ක්‍රම) */

/* 1. Element Selector (සියලුම h1) */
h1 {
  color: #38bdf8;
}

/* 2. Class Selector (class="box" ඇති ඒවා - "." භාවිත කෙරේ) */
.box {
  background-color: black;
  padding: 10px; /* ඇතුළත ඉඩ (Inner space) */
}

/* 3. ID Selector (id="main" ඇති එක - "#" භාවිත කෙරේ) */
#main {
  margin: 20px; /* පිටත ඉඩ (Outer space) */
}
